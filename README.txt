Adds Sharing buttons widget to user specified locations on the site

== Description ==
Adds sharing buttons element with different layouts and positions on different locations in the site, the user can configure it all under settings.

== Installation ==
1. Extract the zip file and just drop the contents in the wp-content/plugins/ directory of your WordPress installation and then activate the Plugin from Plugins page.
2. Go to Settings > WP Sharing Buttons for further options.
